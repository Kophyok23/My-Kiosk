# My Kiosk (Kids Mode Launcher)

My Kiosk သည် ကလေးငယ်များအတွက် ဘေးကင်းလုံခြုံသော ပတ်ဝန်းကျင်ကို ဖန်တီးပေးနိုင်ရန် ရည်ရွယ်၍ ရေးသားထားသော Android Application တစ်ခုဖြစ်သည်။ ဤ App သည် ကလေးများအား မိဘများ ခွင့်ပြုထားသော Apps များကိုသာ အသုံးပြုခွင့်ပေးပြီး ဖုန်း၏ တခြားသော လုံခြုံရေးဆိုင်ရာ Settings များကို မထိခိုက်စေရန် ကန့်သတ်ပေးသည်။

## 🚀 အဓိက လုပ်ဆောင်ချက်များ (Features)
- **App Whitelisting**: မိဘများ ရွေးချယ်ပေးသော Apps များကိုသာ Launcher တွင် ပေါ်စေခြင်း။
- **PIN Protection**: Admin PIN ရိုက်မှသာ Settings များကို ပြင်ဆင်နိုင်ခြင်း။
- **Restrictive Mode**: မလိုလားအပ်သော အကြောင်းအရာများအား ပိတ်ပင်ခြင်း။
- **Simple UI**: ကလေးငယ်များ အသုံးပြုရလွယ်ကူသော Interface ဒီဇိုင်း။
